# Trench Runner 2086 - Jury Guide

## Start here

Double-click `START-HERE.cmd`.

It starts a private local web server and opens the complete submission page. No installation, account or internet connection is required.

## Recommended 90-second review

1. Read the one-line pitch and watch the 60-second trailer.
2. Select `PLAY JURY RUN` or `LAUNCH JURY RUN`.
3. The Jury Run starts automatically with `VANGUARD` and `ACE`.
4. Follow the in-cockpit tutorial. The complete run is capped at 100 seconds.

## Controls

- `W/A/S/D`, arrow keys or `Y/Z`: steer the targeting corridor
- `Space`: fire dual lasers
- `Shift`: boost
- `E`: shield pulse
- `V`: switch between modern neon and retro vector mode
- `Esc`: pause
- Gamepad: left stick, RT fire, RB boost, LB shield

All keyboard controls can be reassigned under `OPTIONEN`. Subtitles are enabled by default. Color-vision palettes and reduced camera movement are available in the same menu.

## What to evaluate

- Fixed-reticle aiming and camera-relative steering
